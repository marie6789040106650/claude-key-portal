安装指导

按照以下步骤配置 Claude Code 使用您的密钥

1. 选择要使用的密钥

2. 选择您的操作系统

3. 配置环境变量

将以下内容添加到您的 shell 配置文件中：

# 添加到 ~/.bashrc 或 ~/.zshrc export ANTHROPIC_BASE_URL="https://claude.just-play.fun/api" export ANTHROPIC_AUTH_TOKEN="sk-ant-api03-...xyz789" export CRS_OAI_KEY="sk-ant-api03-...xyz789"

执行以下命令使配置生效：

source ~/.bashrc 或 source ~/.zshrc
# 添加到 ~/.config/fish/config.fish set -gx ANTHROPIC_BASE_URL "https://claude.just-play.fun/api" set -gx ANTHROPIC_AUTH_TOKEN "sk-ant-api03-...xyz789" set -gx CRS_OAI_KEY "sk-ant-api03-...xyz789"

3. 配置环境变量

在 PowerShell 中运行以下命令：

[System.Environment]::SetEnvironmentVariable("ANTHROPIC_BASE_URL", "https://claude.just-play.fun/api", "User") [System.Environment]::SetEnvironmentVariable("ANTHROPIC_AUTH_TOKEN", "sk-ant-api03-...xyz789", "User") [System.Environment]::SetEnvironmentVariable("CRS_OAI_KEY", "sk-ant-api03-...xyz789", "User")

注意：需要以管理员权限运行 PowerShell

3. 配置环境变量

将以下内容添加到 ~/.bashrc 或 ~/.profile：

export ANTHROPIC_BASE_URL="https://claude.just-play.fun/api" export ANTHROPIC_AUTH_TOKEN="sk-ant-api03-...xyz789" export CRS_OAI_KEY="sk-ant-api03-...xyz789"

执行以下命令使配置生效：

source ~/.bashrc

4. 验证配置

运行以下命令验证环境变量是否配置成功：

echo $ANTHROPIC_BASE_URL echo $ANTHROPIC_AUTH_TOKEN

如果看到正确的URL和密钥，说明配置成功！

5. 测试 Claude Code 连接

打开 Claude Code 并运行一个简单的命令测试连接：

claude "Hello, Claude!"

常见问题

如果连接失败怎么办？

1. 检查环境变量是否正确设置
2. 确认密钥是否处于活跃状态
3. 尝试重启终端或 IDE
4. 查看 详细文档

如何更换密钥？

修改环境变量中的 ANTHROPIC_AUTH_TOKEN 和 CRS_OAI_KEY，然后重启终端即可。

支持哪些 IDE？

VS Code、JetBrains系列（IntelliJ、PyCharm等）、Vim、Emacs 等主流编辑器。
